
# FF Pro Tournament Web App

এটি একটি পূর্ণাঙ্গ ফ্রি ফায়ার টুর্নামেন্ট প্ল্যাটফর্ম।

## GitHub-এ সেটআপ করার নিয়ম:
১. একটি নতুন রিপোজিটরি তৈরি করুন।
২. এই প্রোজেক্টের সব ফাইল (index.html, index.tsx, metadata.json) আপলোড করুন।
৩. GitHub Settings > Pages-এ গিয়ে 'Deploy from branch' থেকে 'main' সিলেক্ট করে সেভ করুন।
৪. আপনার ওয়েবসাইট লাইভ হয়ে যাবে।
